## 🚀 CareerIQ – AI-Powered Resume Analyzer & Career Assistant

### 📌 Overview
CareerIQ is an AI-powered web application that analyzes resumes, extracts skills, maps them to relevant software roles, and provides real-time job opportunities. It also enhances resumes using ATS-based scoring and detects fraudulent job listings.

---

### ✨ Features
- 📄 Resume parsing and skill extraction (238+ skills across 10 categories)  
- 🎯 Role matching engine for 11+ software roles with match scores  
- 🌐 Real-time job aggregation using REST APIs with filtering (location, recency, relevance)  
- 📊 ATS-based resume scoring with automatic FAANG-style rewriting (<70%)  
- 🛡️ Fraud detection system using 30+ trusted domains and 20+ scam patterns  
- 🔒 Fully client-side architecture ensuring privacy and zero data leakage  

---

### 🛠️ Tech Stack
- **Frontend:** HTML, CSS, JavaScript (Vanilla)  
- **Libraries:** PDF.js, Chart.js  
- **APIs:** REST APIs (Remotive)  
- **Tools:** Git, GitHub  

---

### 🏗️ Architecture
Resume → Text Extraction → Skill Matching → Role Mapping → Job Fetching → UI Rendering
